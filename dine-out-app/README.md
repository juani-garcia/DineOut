# DineOutApp

## Dependencies

- Node.js >= 18.16.0

### How to switch to Node.js (18.16.0)

First get nvm (Node version manager)

```bash 
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash
```

Use nvm to get 18.16.0

```bash
nvm install 18.16.0
```

Switch between versions

```bash
nvm use 18.16.0
```
